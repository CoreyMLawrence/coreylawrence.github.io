In my first marketing role, I handled everything from product design to web presence. From Concept to Code

Comfort Airlines: Aircraft Simulator

Reimagining airline management as an immersive, hands-on experience, this simulator lets users navigate ticketing, scheduling, and financial operations effortlessly. Built with accessibility at its core, it allows users to explore the nuances of running an airline—all through a single, intuitive interface. It’s more than a tool; it’s an engaging way to understand real-world aviation logistics.

DevolveAI: AI-Powered Decompiler

DevolveAI makes reverse engineering easy, allowing anyone curious about software development to decompile code. This AI-powered tool turns executables into readable source code, enabling users to explore code analysis effortlessly. With a clean interface, it blends technical depth with intuitive design, helping users of all skill levels learn and customize code with ease.

AdventureAI: Interactive Storytelling

AdventureAI transforms storytelling by combining humor, choice, and AI to create unique, personalized adventures. With each playthrough, AdventureAI crafts a new story based on user choices, guiding players through unexpected twists and lively characters. It’s an endlessly replayable journey, blending AI and narrative for a truly interactive experience.

Phonelo: University Solutions Product Page

The Phonelo site exemplifies my commitment to human-centric design and compelling visual storytelling. Through iteration and user testing, I crafted a responsive web experience that embodies the brand’s innovation and reliability. I used GSAP for advanced animations and created a collapsible custom audio player. These features boost user engagement and conversion rates.

Personal Music Player: Progressive Web App

The music player PWA is optimized for iPhone and iPad, showcasing a responsive design and efficient audio playback. Utilizing advanced JavaScript, it offers dynamic song management and intelligently preloads audio files based on connectivity. Additionally, it supports media session actions for seamless playback control directly from the device’s media interfaces, including lock screens and control centers.

Audio Wave Speech Player: React Component

The Audio Speech Detector is an innovative React component that analyzes audio for speech activity, utilizing the Web Audio API for advanced audio processing. It features a responsive design with engaging visual feedback, including animated icons that react to speech detection. The component implements dynamic audio filtering, noise gating, and energy smoothing to ensure accurate speech recognition.
