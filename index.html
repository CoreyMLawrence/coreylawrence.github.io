From Concept to Code

Comfort Airlines: Aircraft Simulator

Developed an intuitive aircraft simulator that allows users to explore airline operations effortlessly. With a single Python script, users can navigate ticket purchasing, flight scheduling, and financial reporting. This user-friendly tool provides valuable insights into managing an airline while achieving a net profit of over $263M per simulated month.

DevolveAI: AI-Powered Decompiler

Created a web-based AI-driven decompiler that transforms executable files into original source code. With a user-friendly interface, DevolveAI allows users to easily reverse engineer executables, translate the extracted code into multiple programming languages, and customize it with AI enhancements for streamlined development.

AdventureAI: Choose Your Own Adventure

Developed an interactive choose-your-own-adventure game powered by Google's Gemini AI, featuring dynamic storytelling and engaging gameplay. Players navigate unique paths filled with humor and unexpected twists, crafting a different story with each playthrough.
